### 问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Solution 导入失败 | GUID 冲突导致本地化标签唯一键违反 | 改用 C# 代码直接更新 systemform |
| FormXml schema 验证失败 | `cell id` 不是有效 GUID 格式 | 使用 `Guid.NewGuid().ToString("B")` |
| 字段插入到 footer | `LastIndexOf("</rows>")` 匹配到 footer | 改为在第一个 `<control>` 后的 `</row>` 处插入 |
| 字段名不匹配 | 硬编码了不存在的字段名 | 先 `list-fields` 查询实际字段再映射 |
| footer 残留错误字段 | 之前错误插入的字段留在 footer | 新增 `CleanFormFooter()` 方法清理 |
| 实体未找到 | 使用了不存在的实体逻辑名 | 核对解决方案实际实体列表 |
| Program.cs 编译错误 | 方法位置放错导致括号不匹配 | 重新整理类结构 |
| 编译报错 AssemblyInfo 重复 | CheckForm 子项目干扰主项目 | 删除子项目，清理 obj/bin 后重建 |
