# 评分卡类型匹配规则说明

> 创建日期: 2026-06-06
> 说明: 信用分计算Plugin的评分卡类型匹配逻辑，基于D365实际选项集值
> 关联代码: `/Code/Customizations/Plugins/CreditScore/Plugin/CreditScorePlugin.cs`

---

## 评分卡类型概览

PRD定义共 **7种评分卡**：

| 评分卡类型 | categoryId | 适用客户 | 说明 |
|-----------|-----------|---------|------|
| SA级老客户 | 1 | 直销大客户 + 老客户 | S级或A级直销客户，有历史交易 |
| SA级新客户 | 2 | 直销大客户 + 新客户 | S级或A级直销客户，无历史交易 |
| BC级老客户 | 3 | 直销普通客户 + 老客户 | B级或C级直销客户，有历史交易 |
| BC级新客户 | 4 | 直销普通客户 + 新客户 | B级或C级直销客户，无历史交易 |
| 个人客户 | 5 | 个人客户(I级) | 个人客户 |
| 老经销商 | 6 | 经销商 + 老客户 | 有历史交易的经销商 |
| 新经销商 | 7 | 经销商 + 新客户 | 无历史交易的经销商 |

---

## 匹配因子

评分卡类型由 **3个输入因子** 决定：

### 因子1：新老客户

| 判断方式 | 数据来源 | 结果 |
|---------|---------|------|
| 老客户 | `salesorder` 表中存在该客户的订单记录 | `isOldCustomer = true` |
| 新客户 | `salesorder` 表中无该客户订单记录 | `isOldCustomer = false` |

### 因子2：是否经销商

| D365字段 | 值 | 含义 | 结果 |
|---------|-----|------|------|
| `mcs_accountcategory` | **10** | Official Dealer（正式经销商）| `isDealer = true` |
| | **90** | Prospective Dealer（意向经销商）| `isDealer = true` |
| | 其他 | 直销客户 | `isDealer = false` |

### 因子3：直销客户分级

| D365字段 | 值 | 标签 | PRD映射 | 客户级别 |
|---------|-----|------|---------|---------|
| `mcs_accountlevel` | **4** | Diamond | **S级** | 大客户 |
| | **3** | Gold | **A级** | 大客户 |
| | **2** | Silver | **B级** | 普通客户 |
| | **1** | Other | **C级** | 普通客户 |

### 因子4：客户类型（个人客户识别）

| D365字段 | 值 | 标签 | 结果 |
|---------|-----|------|------|
| `mcs_accounttype` | **1** | Individual Account | **个人客户** |
| | **2** | Company Account | 公司客户 |

> ✅ **个人客户识别**: 使用 `mcs_accounttype=1`（Individual Account）识别个人客户，待业务方确认

## 代码实现

### CreditScorePlugin.cs 核心逻辑

```csharp
// 1. 查询Account客户属性
var account = service.Retrieve("account", accountId, 
    new ColumnSet("mcs_accountcategory", "mcs_accountlevel", "mcs_dealerrank"));

// 2. 判断是否经销商
int accountCategory = account.GetAttributeValue<OptionSetValue>("mcs_accountcategory")?.Value ?? 0;
bool isDealer = (accountCategory == 10 || accountCategory == 90);

// 3. 直销客户分级
int accountLevel = account.GetAttributeValue<OptionSetValue>("mcs_accountlevel")?.Value ?? 0;

// 4. 判断新老客户
bool isOldCustomer = CheckHasSalesOrder(service, accountId);

// 5. 匹配评分卡类型
return MatchScoringCardType(isOldCustomer, isDealer, accountLevel, accountCategory, tracer);
```

### MatchScoringCardType 方法

```csharp
private int MatchScoringCardType(bool isOldCustomer, bool isDealer, int accountLevel, int accountCategory, ITracingService tracer)
{
    // 个人客户判断（待业务确认）
    bool isPersonal = (accountCategory == 20 && accountLevel == 1);
    if (isPersonal) return 5;

    // 经销商
    if (isDealer)
    {
        return isOldCustomer ? 6 : 7;
    }

    // 直销客户：SA级 = S(4)或A(3)，BC级 = B(2)或C(1)
    bool isBigAccount = (accountLevel == 4 || accountLevel == 3);
    
    if (isOldCustomer)
    {
        return isBigAccount ? 1 : 3;
    }
    else
    {
        return isBigAccount ? 2 : 4;
    }
}
```
