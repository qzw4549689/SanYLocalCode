## NACE Code 结构说明

NACE (Nomenclature of Economic Activities) 是欧盟经济活动分类标准：

- **第1位数字**：行业大类（Section）
- **第2位数字**：行业分组（Division）
- **更多位数**：更细分的行业类别

## 映射逻辑建议

```csharp
// 伪代码示例
string MapNaceToSanyIndustry(string naceCode)
{
    string prefix = naceCode.Substring(0, 2);
    
    switch (prefix)
    {
        case "01": return "农业";
        case "02": return "林业";
        case var p when p >= "05" && p <= "09": return "矿业";
        case var p when p >= "10" && p <= "33": return "制造业";
        case "23": return "商混";
        case "41":
        case "42": return "建工";
        case "43": return "吊装"; // 需进一步区分施工型/物流型
        case "49":
        case "50":
        case "51": return "集装箱运力";
        case "52": return "港务"; // 需进一步区分港务/物流型吊装
        case "77": return "租赁";
        default: return "其他";
    }
}
```
