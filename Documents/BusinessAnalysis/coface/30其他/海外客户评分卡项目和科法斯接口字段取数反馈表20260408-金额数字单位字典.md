## 数字字段-单位

| 关键字 | 字典说明 | Value | Name |
|--------|---------|-------|------|
| Blank (空值) | 元 | 0 | Blank |
| Thousand | 千元 | 1 | Thousand |
| Million | 百万 | 2 | Million |
| Billion | 十亿 | 3 | Billion |
| Percent | 比例 | 4 | Percent |

---

## JSON 示例

```json
{
  "name": "Net Worth",
  "date": 20241231,
  "fromDate": 20240101,
  "toDate": 20241231,
  "fromAmount": 17345.3550,
  "currency": {
    "value": "EUR",
    "name": "Euro"
  },
  "dimension": {
    "value": "1",
    "name": "Thousand"
  }
}
```

**说明：**
- `fromAmount`: 17345.3550
- `currency`: EUR (欧元)
- `dimension`: Thousand (千元)
- **实际金额** = 17345.3550 × 1000 = **17,345,355 EUR**

---

## 单位换算规则

| dimension.value | dimension.name | 换算倍数 |
|----------------|---------------|---------|
| 0 (或空) | Blank | × 1 (元) |
| 1 | Thousand | × 1,000 (千元) |
| 2 | Million | × 1,000,000 (百万) |
| 3 | Billion | × 1,000,000,000 (十亿) |
| 4 | Percent | 百分比值 |

---

## 关键说明

1. **净资产** 可能的 dimension 包括：空值、Thousand、Million、Billion
2. **财务比率**（如资产负债率）的 dimension 为 Percent
3. 读取 `fromAmount` 时必须同时考虑 `dimension` 和 `currency`
4. 建议统一转换为 **USD** 或 **EUR** 后再进行评分计算

---

## 实际金额计算示例

```csharp
// 伪代码
decimal GetActualAmount(decimal fromAmount, string dimensionValue, string currencyCode)
{
    decimal multiplier = dimensionValue switch
    {
        "0" or null or "" => 1,           // 元
        "1" => 1000,                       // 千元
        "2" => 1000000,                    // 百万
        "3" => 1000000000,                 // 十亿
        "4" => 1,                          // 百分比
        _ => 1
    };
    
    decimal actualAmount = fromAmount * multiplier;
    
    // 再根据 currencyCode 换算为 USD
    // ...
    
    return actualAmount;
}
```
