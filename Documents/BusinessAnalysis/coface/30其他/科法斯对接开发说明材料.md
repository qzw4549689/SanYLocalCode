## 4月第一次对接 Coface

### 需要购买的产品

| 产品 | 说明 | 覆盖指标 |
|------|------|---------|
| **URBA-结构化** | 结构化数据 | 9个指标 |
| **Full Report** | 非结构化数据 | 3个指标 |
| **Full Report CEE** | 中东欧补充产品 | 受限国家（如俄罗斯） |

> 共计接口 2-3 个 API 调用

### 需求分析和接口分析支持

- 三一提供相关结构化量化指标
- 详细数据需要判断（比如诉讼记录）
- Coface 提供取数可行性分析

### 三一 CRM Coface 客户 ID 设置

- 采用**公司名称、税号、注册号、国别**来定位客户
- 可能返回多条记录，让业务重新把公司名称等录入准确
- **最新 Linda 反馈**：三一可提供客户清单数据，Coface 提供相关的 Coface 客户唯一编号
- 可以考虑采用科法斯的**主数据管理**，采用科法斯的数据覆盖
