## 四、JSON路径说明（Coface Cathy邮件）

### 净资产 (Net Assets / Net Worth)
- **JSON路径**: `productDetails.financials.balanceSheet.balanceSheetItems`
- **匹配字段**: `type.value` = 对应国家的indicator编码
- **取值字段**: `fromAmount`
- **示例**:
```json
{
  "type": {
    "value": "1000",
    "name": "Net Worth"
  },
  "date": 20231231,
  "fromDate": 20230101,
  "toDate": 20231231,
  "fromAmount": 43597.8475,
  "currency": {
    "value": "EUR",
    "name": "Euro"
  },
  "dimension": {
    "value": "1",
    "name": "Thousand"
  }
}
```

### 其他3个比率（资产负债率、流动比率、净利润率）
- **JSON路径**: `productDetails.financials.ratios`
- **匹配字段**: `type.value` = 对应国家的indicator编码
- **取值字段**: `fromAmount`
- **示例**:
```json
{
  "type": {
    "value": "4105",
    "name": "Debt ratio"
  },
  "description": "((Borrowing + Current liab.)/Total assets)",
  "date": 20241231,
  "fromDate": 20240101,
  "toDate": 20241231,
  "fromAmount": 0.2526,
  "currency": {}
}
```

### 字段说明

| 字段 | 说明 | 示例 |
|:---|:---|:---|
| `type.value` | **财务科目编码**（有国别差异） | "1000", "4105" |
| `type.name` | 财务科目关键字（有国别差异） | "Net Worth", "Debt ratio" |
| `date` | 财务数据日期 | 20241231 |
| `fromDate` | 起止有效日期-开始 | 20240101 |
| `toDate` | 起止有效日期-结束 | 20241231 |
| `fromAmount` | **财务科目值（金额）** | 17345.3550, 0.2526 |
| `currency.value` | 货币（可配置成USD） | "EUR" |
| `currency.name` | 货币名称 | "Euro" |
| `dimension.value` | 数字单位字典 | "1" |
| `dimension.name` | 数字单位名称 | "Thousand"（千元） |

## 五、开发要点

1. **大部分国家（19个）**使用标准indicator编码：
   - 净资产: indicator(1) - indicator(9) 或 indicator(21)
   - 资产负债率: indicator(9) / indicator(1)
   - 流动比率: indicator(3) / indicator(260)
   - 净利润率: indicator(620) / indicator(7)

2. **印度和马来西亚**使用特殊indicator编码，需单独处理

3. **取数逻辑**: 通过indicator.type.value匹配对应国家的编码

4. **5/22更新**: 可以不用看"Formula"列，直接用indicator.type.value对应编码

5. **JSON取数路径**:
   - 净资产: `productDetails.financials.balanceSheet.balanceSheetItems[].type.value` 匹配 → `fromAmount`
   - 其他比率: `productDetails.financials.ratios[].type.value` 匹配 → `fromAmount`

6. **货币处理**:
   - URBA360财务指标默认EUR，但可配置成USD
   - 需根据currency.value判断是否需要货币转换

7. **单位处理**:
   - dimension.name = "Thousand" 表示千元
   - 需转换为元（×1000）
