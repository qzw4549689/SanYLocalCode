# SolutionViewer

## Purpose
A web-based tool for viewing and exploring D365 solution components.

## Overview
This is a Node.js web application that provides a visual interface for browsing D365 solution metadata and components.

## Key Components
- `public/` - Static web assets
- `server.js` - Main server application
- `package.json` - Node.js dependencies

## Technology
- Node.js
- Express.js
