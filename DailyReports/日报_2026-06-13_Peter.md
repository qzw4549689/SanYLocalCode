### 3. 关键问题与解决

#### 问题 1：弹窗中 `Xrm is not defined`
- **现象**：点击【搜索】报错 `搜索异常: Xrm is not defined`
- **根因**：D365 HTML WebResource 中不支持直接使用 `Xrm` 对象；`ClientGlobalContext.js.aspx` 已在 Modern UI 中废弃且返回 404
- **解决**：弹窗内改用浏览器原生 `fetch` 调用 Web API：
  - `POST /api/data/v9.2/mcs_CofaceSearchCompany` 调用 Custom Action
  - `PATCH /api/data/v9.2/mcs_credit_records({guid})` 更新评估记录
  - `PATCH /api/data/v9.2/accounts({guid})` 更新客户主数据

#### 问题 2：DEV 命令行 `publish-webresource` 被锁
- **现象**：`dotnet run publish-webresource` 报错 `previous [Publish] running`
- **根因**：DEV 环境存在前序 Publish/Solution 作业未释放全局 Publish 锁
- **解决**：用户在 D365 后台通过【保存并发布】单个 WebResource 绕过锁
